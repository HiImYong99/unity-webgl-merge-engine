  Google Play Console 인앱 상품 등록 가이드                                                                                                                               
                                                                                                                                                                          
  전제 조건                                                                                                                                                             
                                                                                                                                                                          
  - Google Play Console에 앱(com.animalpop.app)이 등록되어 있어야 합니다                                                                                                  
  - AAB 파일을 최소 1회 업로드(내부 테스트 트랙 이상)해야 인앱 상품 메뉴가 활성화됩니다

  ---
  Step 1 — AAB 업로드 (처음이라면)

  1. Google Play Console 접속
  2. 앱 선택 → 좌측 메뉴 출시 > 테스트 > 내부 테스트
  3. 새 버전 만들기 → AnimalPop-release.aab 업로드
  4. 저장 후 내부 테스트 트랙에 출시 (검토 없이 즉시 활성화됨)

  이 단계를 거쳐야 인앱 상품 메뉴가 잠금 해제됩니다.

  ---
  Step 2 — 인앱 상품 등록

  1. 좌측 메뉴 수익 창출 > 인앱 상품 클릭
  2. 상품 만들기 버튼 클릭
  3. 아래 정보 입력:

  ┌───────────┬───────────────────────────────────────────┐
  │   항목    │                    값                     │
  ├───────────┼───────────────────────────────────────────┤
  │ 상품 ID   │ remove_ads_hint_pack ← 코드와 반드시 일치 │
  ├───────────┼───────────────────────────────────────────┤
  │ 상품 유형 │ 관리형 상품 (비소모품, 한 번만 구매)      │
  ├───────────┼───────────────────────────────────────────┤
  │ 이름      │ 광고 영구 제거                            │
  ├───────────┼───────────────────────────────────────────┤
  │ 설명      │ 모든 광고를 영구적으로 제거합니다.        │
  ├───────────┼───────────────────────────────────────────┤
  │ 기본 가격 │ 한국 ₩990                                 │
  └───────────┴───────────────────────────────────────────┘

  4. 상태를 활성 으로 변경 후 저장

  ---
  Step 3 — 가격 설정

  1. 상품 편집 화면 하단 가격 섹션
  2. 국가 및 지역 추가 → 대한민국 선택
  3. 가격 990 입력
  4. 다른 국가는 자동 환산 또는 수동 설정

  ---
  Step 4 — 테스트 구매 확인

  실제 결제 없이 테스트하려면:

  1. Play Console → 설정 > 라이선스 테스터 에 테스트 계정 이메일 추가
  2. 해당 계정으로 앱 설치 후 구매 시도 → 결제창에 "테스트 구매" 표시
  3. 실제 결제되지 않고 onPurchaseGranted 콜백 정상 발동

  ---
  주의사항

  - 상품 ID는 한 번 등록하면 변경/삭제 불가 → remove_ads_hint_pack 그대로 사용
  - 비소모품이므로 한 계정당 1회만 결제됨 (코드의 acknowledgePurchase + restorePurchases 로 복원 처리됨)
  - 앱 첫 출시 전 상품을 미리 등록해 두면 심사 시 함께 검토됨
