# 프로젝트 개요
- Unity 2022.3.62f3 WebGL 빌드를 Android 웹뷰(래퍼)로 마이그레이션 중.
- 광고(AdMob)만 포함되며 IAP(인앱결제)는 완벽히 배제함.

# 게임 코어 로직 규칙 (절대 수정 금지)
- 레벨별 에셋고정.
- 게임 오버 판정: 불규칙한 모양의 에셋이 단순히 선을 넘는 것이 아니라, 컨테이너 밖으로 완전히 떨어져 넘쳤을 때(Overflow)만 게임 오버 처리할 것. (너무 빨리 끝나는 현상 방지)
- 점수판 데이터: '디저트 이름', '토스 점수', '비고' 컬럼은 사용하지 않음.

# 코딩 스타일 및 환경
- 웹뷰 통신: Unity-JS 브릿지를 통한 양방향 통신 코드를 작성할 때는 항상 비동기 처리를 고려할 것.
